## 개선 권고 4가지

**1. night-worker에서 E2E/환경의존 태스크 제외**
- TASK-190 같은 $5 낭비 방지
- config.json의 `nightWorker.types`에서 e2e 명시적 제외

**2. 세션 수 기반 circuit breaker 추가**
- 현재: 비용 $5 초과 시 중단
- 추가: 세션 6회 초과 시에도 중단
- TASK-203 (12세션 $10.75)을 6세션 $5 정도에서 끊을 수 있었음

**3. review reject 사유를 다음 task 세션에 전달**
- 현재: reject 후 처음부터 다시 시도
- 개선: reject 이유를 프롬프트에 포함 → 같은 실수 반복 방지

**4. 넓은 scope 태스크에 예상 비용 경고**
- "전수 탐색", "UX 흐름 검증" 같은 키워드 감지 시 비용 경고 표시
- scope 파일이 10개 초과면 예상 비용 상향 안내
