# PRD-001: 오케스트레이션 대시보드

## 목표

AI 에이전트 기반 태스크 오케스트레이션 시스템의 실행 상태를 실시간으로 모니터링하고 관리할 수 있는 대시보드를 구축한다.

## 기능

- Task 워터폴 뷰 (의존 관계 시각화)
- Sprint 목록 및 상세 뷰
- Plan 트리 뷰
- 비용 모니터링 (토큰, 비용, 시간 추적)
- 터미널 뷰 (실시간 실행 로그)

## 서버 설정

- 포트 번호는 환경변수(PORT)로 받는다.
